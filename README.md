## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/sg-films.git
   ```

2. **Install dependencies**
   ```bash
   cd sg-films
   npm install
   ```

3. **Run the development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   ```
   http://localhost:3000
